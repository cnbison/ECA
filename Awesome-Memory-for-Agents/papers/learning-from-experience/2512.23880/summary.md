# CASCADE: Cumulative Agentic Skill Creation through Autonomous Development and Evolution

**arXiv:** [2512.23880](https://arxiv.org/abs/2512.23880)
**PDF:** [下载](https://arxiv.org/pdf/2512.23880) · 本地: `2512.23880.pdf`
**作者:** Huang, Xu, Chen, Junwu, Fei, Yuxing, Li, Zhuohan, Schwaller, Philippe, Ceder, Gerbrand
**日期:** 2025/12/29

## Abstract

Large language model (LLM) agents currently depend on predefined tools or early-stage tool generation, limiting their adaptability and scalability to complex scientific tasks. We introduce CASCADE, a self-evolving agentic framework representing an early instantiation of the transition from &#34;LLM + tool use&#34; to &#34;LLM + skill acquisition&#34;. CASCADE enables agents to master complex external tools and codify knowledge through two meta-skills: continuous learning via web search, code extraction, and memory utilization; self-reflection via introspection, knowledge graph exploration, and others. We evaluate CASCADE on SciSkillBench, a benchmark of 116 materials science and chemistry research tasks. CASCADE achieves a 93.3% success rate using GPT-5, compared to 35.4% without evolution mechanisms. We further demonstrate real-world applications in computational analysis, autonomous laboratory experiments, and selective reproduction of published papers. Along with human-agent collaboration and memory consolidation, CASCADE accumulates executable skills that can be shared across agents and scientists, moving toward scalable AI-assisted scientific research.
